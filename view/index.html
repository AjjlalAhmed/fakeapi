{ fakeApi }

Fake Online REST API for Testing and Prototyping

try it

how it works

fetch('https://jfakeapi.herokuapp.com/api/comments?limit=1')
            .then(response => response.json())
            .then(json => console.log(json))

how to use it

FakeApi can be used to fetch JsonData for your website. you can use examples below to check how fakeApi works and feel free to enjoy it in your awesome projects!

resources

FakeApi comes with a set of 3 common resources:

  • /posts 100 posts
  • /comments 500 comments
  • /todos 200 todos

You can add additional query

  • userId todos & posts
  • postId comments
  • limit 1 to 100

routes

Base Data URL

https://jfakeapi.herokuapp.com/api/

All HTTP methods are supported

  • GET /todos
  • POST /todos
  • PUT /todos
  • PATCH /todos
  • DELETE /todos
